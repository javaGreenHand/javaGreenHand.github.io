# 配置文件

## schema.xml
具体结构如下：
```
    <schema>
      <types>
      <fields>
      <uniqueKey>
      <copyField>
    </schema>
```
其中`types`和`fields`是可选的，具体为`field`、`dynamicField`、`copyField`、`fieldType`。

1. `fieldType`：字段类型
2. `field`：普通字段
3. `dynamicField`：动态字段
4. `copyField`：复制字段
